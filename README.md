### Templating Engines

#### Use normal HTML and plain JavaScript in your templates

    - EJS ===           <p><%= name %></p>;

#### Use minimal HTML and custom template language

    - Pug (Jade) ===    p #{name} ;
    - Handlebars ===    <p>{{ name }}</p>
